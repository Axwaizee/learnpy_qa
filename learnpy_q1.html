Python Questions

Basic Print & Input

  1. Write a program that prints your name.
  2. Ask the user for their favorite color and print it back.
  3. Print three lines of text using a single print() statement.
  4. Ask the user for their name and age, then print a short message like “Hello Sam, you are 15 years old.”
  5. Print the result of adding two numbers entered by the user.

Strings

  1. Ask the user for a word and print it in uppercase.
  2. Ask the user for a sentence and print how many characters it has.
  3. Ask the user for their first and last name, then print their full name in reverse order.
  4. Ask for a sentence and print only the first five characters.
  5. Ask for a name and check if it contains the letter "a".

Numbers & Basic Operations

  1. Ask the user for two numbers and print their sum, difference, and product.
  2. Ask the user for a number and print double that number.
  3. Ask for a number and print whether it’s even or odd.
  4. Ask the user for a number and print its square.
  5. Ask for two numbers and print which one is bigger.

If / Else Statements

  1. Ask for the user’s age and print “You are an adult” if they’re 18 or older, otherwise “You are a minor.”
  2. Ask for a password and print “Access granted” if it matches a preset password.
  3. Ask for a temperature and print if it’s hot, cold, or moderate.
  4. Ask the user for a grade (A–F) and print a short comment based on it.
  5. Ask if it’s raining (yes/no) and print whether the user should take an umbrella.

Loops

  1. Print numbers from 1 to 10 using a for loop.
  2. Ask the user for a number and print all numbers from 1 up to that number.
  3. Ask for a word and print each letter on a new line.
  4. Use a while loop to ask the user for a password until they enter it correctly.
  5. Use a loop to print the word “Python” 5 times.

Lists

  1. Create a list of 5 fruits and print each one.
  2. Ask the user to enter 3 favorite movies, store them in a list, and print the list.
  3. Given a list of numbers, print only the even ones.
  4. Add a new item to a list and print the updated list.
  5. Ask the user for a fruit and check if it’s in your list of fruits.

Functions

  1. Write a function that prints “Hello!” when called.
  2. Write a function that takes a name and prints “Hello, [name]”.
  3. Write a function that adds two numbers and returns the result.
  4. Write a function that checks if a number is even.
  5. Write a function that prints the length of any word given as input.

Bonus

  1. Ask for someone’s birth year and calculate their age.
  2. Create a simple calculator that can add, subtract, multiply, or divide two numbers.
  3. Ask the user for 5 names, store them in a list, then print them alphabetically.
  4. Ask for a sentence and count how many times a specific word appears.
  5. Make a simple “guess the number” game (between 1 and 10).
